# Morphism Systems

Trust infrastructure for AI software delivery.

Morphism is a governance framework that uses category theory -- sheaf cohomology for drift detection, Banach contraction for convergence, and information-theoretic entropy for compliance measurement -- to keep AI-assisted engineering consistent, auditable, and governable.

## Quick Start (Free tier)

```bash
npm install -g @morphism-systems/cli

morphism init        # Scaffold .morphism/config.json
morphism validate    # Run governance validation pipeline
morphism score       # Print maturity score breakdown
morphism doctor      # Check environment and prerequisites
```

## What we build

| Component | Description |
|-----------|-------------|
| **CLI** | `morphism init`, `morphism validate`, `morphism score` for developer workflows |
| **MCP server** | Expose governance tools to Claude, Cursor, and other AI agents |
| **Web dashboard** | Real-time kappa, entropy trends, drift status, and audit trail |
| **SDK** | Embeddable governance client for third-party platforms |
| **Governance engine** | Python core with sheaf-theoretic drift detection and contraction-based healing |
